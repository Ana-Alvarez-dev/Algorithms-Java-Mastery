# Primary Textbooks

## Introduction to Algorithms

**Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, Clifford Stein**

MIT Press

Commonly known as **CLRS**, this is one of the most influential books in computer science.

Relevant topics:

- algorithm analysis;
- computational complexity;
- searching;
- sorting;
- asymptotic notation;
- correctness proofs;
- algorithm design.

This book provides the theoretical foundation adopted throughout the repository.

# Citation Philosophy

Throughout this repository, concepts should always be studied from authoritative sources rather than isolated code examples.

The recommended learning progression is:

```text
Academic Theory
        ↓
Algorithm Design
        ↓
Java Implementation
        ↓
Testing
        ↓
Benchmarking
        ↓
Engineering Reflection
```

This methodology ensures that implementations are grounded in sound theoretical principles and recognised engineering practices.
