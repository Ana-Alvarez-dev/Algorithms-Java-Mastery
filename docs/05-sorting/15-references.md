# References

## Purpose

This document provides the academic and technical references used throughout the **Sorting** module.

The objective of this bibliography is not merely to cite sources, but to demonstrate that the concepts presented in this module are grounded in well-established literature from computer science, algorithm analysis, software engineering, and the Java ecosystem.

Whenever possible, primary references have been preferred over secondary material.

The books listed below are internationally recognized and widely used in undergraduate and graduate Computer Science programs.

# Primary References

## Introduction to Algorithms

**Thomas H. Cormen**  
**Charles E. Leiserson**  
**Ronald L. Rivest**  
**Clifford Stein**

> *Introduction to Algorithms (4th Edition).*  
MIT Press, 2022.

This is one of the most influential algorithm textbooks ever published.

Throughout this module it provides the theoretical foundations for:

- algorithm analysis;
- asymptotic complexity;
- divide-and-conquer algorithms;
- Merge Sort;
- correctness reasoning;
- recurrence relations;
- lower bounds for comparison sorting.

**Donald E. Knuth**
